# PROJECT-CONTEXT

- **Project Name**: ORBIT
- **Goal**: Open-source Work OS
- **Tech Stack**: Next.js 16 (pnpm latest), React 19, TypeScript, Tailwind, Supabase, Drizzle, Turborepo, pnpm
- **Architecture**: Microkernel + Plugin System
- **5 Core Services**:
  1. **Data Engine**: Responsible for data storage and management.
  2. **View Renderer**: Responsible for UI display and interaction.
  3. **Permission System**: Responsible for access control and security.
  4. **Plugin Runtime**: Responsible for plugin execution and lifecycle.
  5. **API Layer**: Responsible for external communication and integration.
